# **App Name**: Auto-Yum

## Core Features:

- AI Recipe Discovery: Leverage OpenAI to parse user queries based on ingredients, health goals, diet preferences, and time limits, then formulate queries for the search engine.
- Algolia-Powered Search: Implement Algolia for instant recipe search results, with advanced filtering options for diet, prep time, and ingredients.
- Recipe Details Display: Display detailed recipe information fetched from the Supabase database, including ingredients, instructions, and nutritional information. Link into Algolia index.
- AI Cooking Assistant: Provide a recipe-aware AI tool that offers ingredient substitutions, cooking tips, and health insights via OpenAI.
- User Authentication: Implement user accounts using Supabase Auth to enable saving favorite recipes and dietary preferences.

## Style Guidelines:

- Primary color: Vibrant orange (#FF8C00) to represent the energy and health benefits of nutritious food.
- Background color: Light orange (#FFF3E0), a desaturated shade of the primary, providing a warm and inviting backdrop.
- Accent color: Yellow-orange (#FFA500), an analogous color to the primary, used to highlight key actions and elements.
- Headline font: 'Belleza' sans-serif for headings and prominent text elements.
- Body font: 'Alegreya' serif font for readability in recipe descriptions and details.
- Use clean, modern icons to represent different food categories, dietary restrictions, and cooking actions.
- Design a mobile-first, responsive layout that ensures readability and ease of navigation on all devices.
- Incorporate subtle transitions and animations to provide feedback on user interactions and enhance the overall experience.
